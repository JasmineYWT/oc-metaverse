# 🏠 OC Metaverse - 你的 OC 社交元宇宙

> 一个完全在浏览器中运行的 OC（原创角色）社交模拟器。  
> 你可以创建角色、群聊、发动态、逛论坛，所有角色由 AI 驱动，拥有独立性格和记忆。

---

## ✨ 功能一览

### 💬 消息
- 单聊 & 群聊，支持文字、表情、图片/视频卡片、转账、红包
- AI 角色会根据性格、世界观、配对关系自主回复
- 引用回复、多选转发、消息合并、聊天记录回溯

### 📱 论坛
- 发帖（文字 + 图片 + 标签），点赞评论回复
- OC、NPC、写手太太均可自动发帖互动
- 热搜榜、标签筛选、全文搜索

### 🌐 动态
- 朋友圈风格，支持文字/音乐/心情/任务卡片
- OC 会自动点赞评论转发，形成社交互动
- 动态内容同步更新到角色的心情日记、音乐记录

### 👥 联系人
- 按字母排序，群聊 & 单人分组
- 支持左滑删除、创建新角色/群聊

### 🎛️ 妈咪中心（全局控制台）
- 个人设置、世界书管理、API 配置
- 表情分组管理、自动生成频率设置
- 自定义主题、字体、气泡样式
- 数据导出/导入

## 🚀 如何使用

### 1. 打开网页
直接访问：**https://JasmineYWT.github.io/oc-metaverse/**

### 2. 配置 AI（必须）
1. 点击左上角头像进入「妈咪中心」
2. 切换到「API 配置」标签
3. 填写你的 API URL、API Key、模型名称（需支持 OpenAI 格式接口）
4. 点击「测试连接」确认可用

### 3. 开始玩
- 点击底部「消息」，与你的 OC 聊天
- 在「论坛」发帖、刷帖
- 在「动态」看 OC 们的朋友圈
- 所有数据保存在浏览器本地，不会丢失

## 🛠️ 技术说明

- **纯前端**：HTML + CSS + JavaScript，无后端依赖
- **数据存储**：localStorage 全本地持久化
- **AI 接口**：兼容 OpenAI Chat Completions 格式
- **部署**：GitHub Pages / Netlify / 任何静态托管

---

## ⚠️ 隐私提示

- 你的 API Key 仅保存在本地浏览器中，**不会上传到任何服务器**
- 聊天记录、角色设定等数据也全部存储在你自己的浏览器里
- 换设备或清空浏览器缓存会导致数据丢失，请定期在妈咪中心「数据管理」导出备份
